ranking.php API リファレンス

ランキングの取得

パラメータ 型 値 メソッド
action (必須) string ranking (固定) GET
gameName (必須) string ゲーム名
半角英数字 10 文字以内 		POST
total integer 10 (デフォルト) POST

action (必須) string add (固定) GET
gameName (必須) string ゲーム名
半角英数字 10 文字以内 		POST
userName (必須) string ユーザ名
半角英数字 10 文字以内 		POST
score (必須) integer スコア POST
token (必須) string トークン

を連結した文字列の MD5 ハッシュ値。 		POST
